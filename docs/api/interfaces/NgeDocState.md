# NgeDocState

`interface`

Representation of the documentation state.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `currLink?` | `NgeDocLink` | Current active link. |
| `links` | `NgeDocLink[]` | Root links of the site. |
| `meta` | `NgeDocMeta` | Metadata informations about the documentation. |
| `nextLink?` | `NgeDocLink` | Next link of the current link. |
| `prevLink?` | `NgeDocLink` | Previous link of the current link. |
