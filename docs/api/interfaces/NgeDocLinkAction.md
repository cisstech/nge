# NgeDocLinkAction

`interface`

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `icon?` | `NgeDocIcon` | Icon to render for the action. |
| `run` | `NgeDocLinkActionHandler` | Action handler. (A string value here means that the action is an url to open in a new tab) |
| `title?` | `string` | Title of the action. |
| `tooltip?` | `string` | Action tooltip |
