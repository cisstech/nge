# NgeDocBrand

`interface`

The header brand: one logo and wordmark shared by every site.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `href?` | `string` | Where clicking the brand navigates (an Angular routerLink). Defaults to `/`. |
| `icon?` | `NgeDocIcon` | Optional logo icon. |
| `title` | `string` | Wordmark shown next to the logo. |
