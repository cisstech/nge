# NgeDocNavLink

`interface`

A top-level navigation link shown in the theme header.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `external?` | `boolean` | Open in a new tab as a plain anchor instead of routing internally. |
| `href` | `string` | Target url (an Angular routerLink, or an absolute url when `external`). |
| `icon?` | `NgeDocIcon` | Optional icon. |
| `title` | `string` | Text of the link. |
