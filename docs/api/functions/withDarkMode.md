# withDarkMode

`function`

Set the default color scheme applied before the user picks one.

## Signature

```typescript
function withDarkMode(mode: NgeDocColorScheme): NgeDocFeature
```

### Parameters

- `mode` (`NgeDocColorScheme`) - `auto` (follow the OS, default), `dark` or `light`.

### Returns

`NgeDocFeature`
